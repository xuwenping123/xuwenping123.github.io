hexo文件介绍

完成安装hexo后，所选择的文件夹内会自动生成些文件，文件列表如下：

_config.yml
配置文件，关于hexo的title、author、主题、部署路径等均在此文件中配置

package.json
应用信息描述文件，以json格式描述当前安装的基于hexo的插件，除了默认安装的几个插件以外，还可选择安装如部署到 github 上的hexo-deployer-git，站点地图插件 hexo-generator-sitemap、hexo-generator-baidu-sitemap等。

scaffolds
模板文件夹，新建文章时，会基于模板创建文件。如，可在此文件夹内post.md文件中统一定义新建文章时的头部格式。

source
资源文件夹，用户博客上展示的内容（除却配置）均保存在此文件夹内，如，新建的文章保存在 source/_posts 下，新建的标签也在 source/tags 下。

themes
主题文件夹，在_config.yml文件中配置博客所使用主题，主题文件夹包括主题下的详细设置文件与样式控制文件等。
